## 三、分析框架

### 3.1 三维分析链条

```
政策工具组合 → 组织与治理结构 → 培养与能力生成机制
```

注意：**用"分析链条"，不用"因果链"**。

原因：韩国和台湾案例存在反向影响（培养安排和组织结构对政策工具选择有反馈），不是严格单向因果。

### 3.2 四维操作化

"战略性重构"操作化为四个观测维度：

1. 治理权重再分配（A）
2. 资源配置与平台化（B）
3. 培养逻辑技术路线化（C）
4. 国际嵌入与流动（D）

每维 0/1/2 三档编码，参见正文表 2。

### 3.3 五案例 + 三组解释变量

**案例**（四个主案例 + 一个对照案例）：

- 美国（主）
- 日本（主）
- 韩国（主）
- 中国台湾地区（主）
- 欧盟（对照——因教育主权分散）

**三组解释变量**（解释路径分化）：

- 产业结构与价值链位置
- 政策工具组合与执行能力
- 高校治理与技能形成制度

### 3.4 五种政策路径

四个主案例 + 欧盟对照，对应五种路径：

| 案例 | 路径 | 英文 |
|---|---|---|
| 美国 | 法案约束 / 条件化约束 | legislative conditionality |
| 日本 | 行政动员 + 法案嵌套 | administrative mobilization |
| 韩国 | 行政动员 + 财政集中 | fiscal concentration |
| 中国台湾地区 | 专项立法授权 | special legislative authorization |
| 欧盟（对照） | 超国家协调 | supranational coordination |

## 九、协作偏好（您与 Claude 的固化模式）

经过 30+ 轮验证的协作节奏：

1. **审稿先于动稿**：提出修改清单 → 等用户确认 → 才动文件
2. **多专家透镜**：每轮自评切换"资深半导体专家 + 工程教育专家 + 科技政策专家 + 期刊主编"
3. **P0/P1/P2 分层**：所有问题按优先级排序，附预估工作量
4. **可直接替换的文本**：每条修改给原文 + 改后 + 理由
5. **不编造文献**：不确定信息标"可能不准确，建议核实"
6. **B 稿评审建议批判性采纳**：稀释学术品质的建议明确拒绝
7. **脚本化操作**：参考文献重编号、禁用词扫描、引而未列检查全部脚本化
8. **不擅自重写**：重大改动前问，不直接给"重写版"
