# C. 政治措辞合规

中国 C 刊编辑部对政治措辞的把关极严。任何一处违规都会被退回，严重时直接退稿。本文件覆盖学术写作中的所有敏感点。

---

## 一、涉台措辞（最高敏感度）

### 1.1 基本原则

- "台湾"**不**作为独立国家与其他国家并列
- 与国家并列时必用"中国台湾地区"或"中国台湾"
- 涉及当局机构、法规时需特殊处理

### 1.2 标准用语对照

| 错误表述 | 正确表述 | 说明 |
|---|---|---|
| 台湾 | 中国台湾地区 / 中国台湾 | 与国家并列时 |
| 台湾政府 | 台湾地区有关方面 / 台湾当局 | 视语境 |
| 台湾教育部 | 台湾地区相关主管部门 | 不用"教育部" |
| 台湾工业局 | 台湾地区行政管理机构 | 不用"工业局" |
| 中台关系 | 两岸关系 | 标准用语 |
| 台湾的国家创新政策 | 台湾地区的创新政策 | 删"国家"前缀 |
| 中国大陆与台湾 | 大陆与台湾地区 / 祖国大陆与台湾地区 | 视语境 |
| 中华民国 / 中华民国政府 | 绝不使用 | 政治禁忌 |

### 1.3 跨经济体比较中的特殊处理

跨国比较研究中，台湾不能作为独立国家与美国、日本、韩国等并列。处理方法：

**方法 1**：用"经济体"概括：

```
本研究比较美国、日本、韩国等经济体以及中国台湾地区在 X 议题上的政策路径。
```

**方法 2**：分组：

```
本研究比较四个主案例——美国、日本、韩国与中国台湾地区，并以欧盟为对照案例。
```

注意"中国台湾地区"列于其他国家之后，分组表述。

**方法 3**：脚注说明：

```
注：本文所涉"台湾"均指中国台湾地区。
```

### 1.4 表格中的处理

```
表 1：四案例 + 对照案例的政策工具对比

| 案例 | 政策工具 | ... |
|---|---|---|
| 美国 | ... | ... |
| 日本 | ... | ... |
| 韩国 | ... | ... |
| 中国台湾地区 | ... | ... |
| 欧盟（对照） | ... | ... |
```

不写"台湾"作为独立列项。

### 1.5 涉台法规的处理

台湾地区当局发布的法规、政策文件，处理规则：

1. **加引号**：法规名前后加中文引号
2. **去"国家"**：若法规原名含"国家"二字，必须删除或替换
3. **颁布主体改写**：不用"台湾教育部""台湾工业局"等具体机构名

示例：

```
原名：《国家重点领域产学合作及人才培育创新条例》
正确引用：台湾地区"重点领域产学合作及人才培育创新条例"

错误：
[20] 台湾教育部. 国家重点领域产学合作及人才培育创新条例[Z/OL].

正确：
[20] 台湾地区相关主管部门. "重点领域产学合作及人才培育创新条例"[Z/OL].
(2021)[2026-03-01]. https://...
```

### 1.6 英文表述

| 错误英文 | 正确英文 |
|---|---|
| Taiwan | China's Taiwan region |
| Taiwan government | Taiwan authorities / Taiwan's relevant authorities |
| Taiwan Ministry of Education | Relevant authorities of Taiwan region |
| Republic of China / ROC | 绝不使用 |
| Taiwanese | Of Taiwan region |

英文摘要中：

```
正确：This paper compares four principal cases—the United States, Japan,
South Korea, and China's Taiwan region—with the European Union as a
contrast case.
```

### 1.7 机构英文名

涉台机构的英文名也需调整：

- TSMC：可直接用"TSMC"或"台积电"，无须特殊处理（公司名）
- 台大、台湾大学：可用"National Taiwan University"（既有英文名）但中文用"台湾大学"
- TSRI：可用"Taiwan Semiconductor Research Institute"（既有英文名）

公司、大学等具体机构英文名保留原称即可，不需改"China's Taiwan region"前缀。但行政机构必须避免"Taiwan + 部 / 局 / 院"的组合。

## 二、涉港澳措辞

### 2.1 基本原则

| 表述 | 适用场景 |
|---|---|
| 香港 | 一般语境（学术语境下通常可简称）|
| 中国香港 / 中国香港特别行政区 | 正式文件、与国家并列时 |
| 香港特区 | 简称 |
| 澳门 / 中国澳门 / 澳门特别行政区 | 同上 |

跨国比较研究中，香港作为独立经济体出现时用"中国香港地区"或"中国香港特别行政区"。

### 2.2 涉港机构

- 香港大学、香港中文大学等大学：保持原名
- 香港特区政府：可用 "Hong Kong SAR Government" 或 "香港特区政府"
- 不用"香港政府"作为独立政体

---

## 三、其他涉土地敏感点

### 3.1 南海问题

- 涉及南海岛礁时使用中国官方命名（西沙、南沙、东沙、中沙）
- 不使用"Spratly""Paracel"等非中文命名做主称
- 不引用立场偏颇的国际法庭裁决

### 3.2 西藏 / 新疆

- 用"西藏自治区""新疆维吾尔自治区"
- 不用境外某些刊物使用的替代命名
- 涉及民族议题时避免歧视性表述

### 3.3 钓鱼岛

- 中文用"钓鱼岛"
- 英文用"Diaoyu Islands"
- 不用日方称谓"尖閣諸島"或"Senkaku"

## 四、涉外措辞

### 4.1 国家名称

| 正确 | 错误 |
|---|---|
| 美国 / 美利坚合众国 | / |
| 日本 | / |
| 韩国 / 大韩民国 | 南朝鲜（已废用）|
| 朝鲜 / 朝鲜民主主义人民共和国 | 北朝鲜（已废用）|
| 俄罗斯 / 俄罗斯联邦 | 苏联（除历史语境）|

### 4.2 跨国机构

- 欧盟 / European Union / EU
- 北约 / NATO
- 联合国 / United Nations / UN
- 经合组织 / OECD

机构全称首次出现时给出，后续用简称。

---

## 五、历史事件

涉及近现代史敏感事件时：

- 客观陈述，不带政治评价
- 引用权威学术研究，不用网络流言
- 不展开敏感时段（如 1989 等）
- 引用历史数据时核对官方统计口径

---

## 六、宗教与民族

- 用规范术语（如"伊斯兰教"不用"回教"）
- 涉及民族议题时用规范民族名（"维吾尔族"不用"维族"）
- 避免可能被解读为歧视的表述

---

## 七、政策表述

### 7.1 引用领导人讲话

- 引用最新表述（避免引用已过时的版本）
- 用官方权威来源（《人民日报》、新华社）
- 不擅自解读或缩写

### 7.2 引用政策文件

- 用文件全称
- 标注发布机构（中共中央、国务院等）
- 引用最新版本

### 7.3 政策评价的尺度

学术论文可以分析政策的成效与问题，但需注意：

- 评价基于实证证据，不基于个人观点
- 用学术语言（"政策实施面临 X 挑战"）而非政治化语言（"政策失败"）
- 避免对中央政策的根本性否定
- 对地方政策的批评应有依据

## 八、扫描脚本

```bash
#!/bin/bash
# 政治敏感词扫描

FILE=$1

echo "=== 涉台高风险表述 ==="
grep -nE '台湾政府|台湾的国家|台湾教育部|台湾工业局|中华民国' "$FILE"

echo "=== 涉台需检查（独立"台湾"可能违规）==="
grep -nE '(?<![国国国地区])台湾(?![地区大学])' "$FILE" | head -20

echo "=== 涉港澳需检查 ==="
grep -nE '香港政府|澳门政府' "$FILE"

echo "=== 已废用术语 ==="
grep -nE '南朝鲜|北朝鲜|回教|维族' "$FILE"

echo "=== 不规范敏感岛礁命名 ==="
grep -nE 'Spratly|Paracel|Senkaku|尖閣' "$FILE"
```

保存为 `scan-political.sh`，每轮修订后跑一次。

## 九、政治表述检查的频率

| 阶段 | 频率 |
|---|---|
| 初稿 | 每章写完时扫一次 |
| 中期修改 | 每 2-3 轮扫一次 |
| 投稿前 | 必扫，且需人工通读高风险段落 |

人工通读尤其重要——脚本只能扫已知模式，新表述需人工判断。

---

## 十、不确定时的处理

如果不确定某表述是否合规：

1. 不要冒险——改用更稳妥的表述
2. 查询权威源：人民日报、新华社、国务院新闻办的同类表述
3. 询问期刊编辑：投稿前可咨询期刊编辑

宁可保守，不要冒进。一次政治措辞失误可能导致整篇论文被毙，且对作者声誉有长期影响。

---

## 十一、本节适用边界

本节规则适用于：

- 中国大陆 C 刊投稿
- 中文政策报告
- 面向国内读者的学术著作

不适用于：

- 海外英文期刊投稿（按目标期刊编辑政策）
- 港澳台地区期刊投稿
- 国际会议论文（按会议指南）

但如果作者计划同一篇论文先投国内 C 刊后改投海外期刊，建议**先按国内 C 刊标准写**，改投时再针对性调整。反过来调整代价更大。
