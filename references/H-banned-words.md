# H. 禁用词与密度管理词完整清单

本文件为词汇层面的硬清单，配合 `A-style-guide.md` 使用。修改稿件时直接 grep 这些词，按表替换。

---

## 一、硬禁用词（不可出现，无例外）

| 词 | 类型 | 典型误用 | 替代方案 |
|---|---|---|---|
| 深度 | 程度副词 | 深度参与、深度介入、深度重构、深度赋能 | 实质性、直接、前置、广泛 |
| 显著 | 程度形容词 | 显著提升、效果显著 | 大幅、明显改善（注：明显也需控密度）、可观察到的 |
| 赋能 | 动词 | 数字赋能教育、技术赋能产业 | 推动、支持、强化、为……提供条件 |
| 恰恰 | 副词 | 恰恰相反、恰恰说明 | 反过来看、这表明 |
| 重塑 | 动词 | 重塑格局、重塑生态 | 改变、重构、改写 |

例外：以下情况允许保留：
- "深度学习"（专有名词）
- "深度"在技术含义中（如"测量深度")
- "统计显著性"（统计学专用术语）
- "重塑力"作为名词概念（罕见）

## 二、密度管理词（控密度，不消灭）

### 2.1 而非

- 上限：全文 ≤ 5 处
- 超出时按以下替换：

| 替代 | 适用语境 |
|---|---|
| 而不是 | 一般替换 |
| 并非 | 强否定 |
| 不属于 | 类别判断 |
| 区别于 | 对比强调 |

避免在同一句内"而非……而非……"双连。

### 2.2 不仅……更……

- 上限：全文 ≤ 2 处
- 替代：

| 替代 | 节奏 |
|---|---|
| 不仅……也…… | 平稳并列 |
| 既……又…… | 双重判断 |
| ……同时…… | 时间并存 |
| ……更进一步…… | 时序递进 |

### 2.3 从 A 到 B

- 上限：全文 ≤ 2 处
- 同构句式连续出现时打散：

| 反例 | 改写 |
|---|---|
| 从分散到统筹 | 由分散转入统筹 |
| 从单一改革到系统重构 | 经单一改革推进至系统重构 |
| 从院校内部转向国家主导 | 治理重心由院校内部移至国家层面 |

### 2.4 这一

- 上限：控密度，不连续段落重复
- 替代：该、此、上述、前述、所述
- 注意：避免"这一过程""这一现象""这一变化"在同一段三次以上

### 2.5 硬约束

- 正文谨慎，表格内允许
- 替代：法案约束、条件化约束、刚性约束
- 跨国比较语境推荐"条件化约束"以对应英文 conditionality

### 2.6 明显

- 控密度，全文 ≤ 5 处
- 替代：可观察到的、大幅、突出、（用具体数据替代）

### 2.7 系统性 / 全面性 / 深层次

- 作为定语前缀禁用
- "系统性重构"→"结构性重构"或具体描述
- "全面性推进"→"在 X、Y、Z 三个方面同时推进"
- "深层次原因"→"结构性原因"或具体说明

## 三、句首禁语清单

以下短语**绝不**作为句子起首：

```
值得注意的是
值得一提的是
值得关注的是
不难发现
不难看出
显而易见
显然
毋庸置疑
不可否认
众所周知
一言以蔽之
总而言之
总的来说
综上所述
综合以上分析
归根结底
归结而言
由此可见
不言而喻
可以这样说
有理由相信
```

替换原则：**直接给出判断句**，不需要任何引导语。

## 四、套话短语清单

下列短语在任何位置都尽量避免：

```
具有重要意义
具有重要价值
具有深远影响
具有重要的理论价值与实践意义
在新时代背景下
在新的历史条件下
为……提供了重要参考 / 提供了有益借鉴
是……的应有之义
是……的必然要求
势在必行
迫在眉睫
前所未有
当务之急
可圈可点
方兴未艾
应运而生
亟待加强
日益凸显（替代：突出、加强、密集）
持续凸显
深度交织 / 深度嵌合
全方位 / 多维度 / 多层次（作前缀时）
```

替换原则：要么删除（多数情况），要么改写为具体陈述。

## 五、虚化动词清单

应替换为实义动词：

| 反例 | 正例 |
|---|---|
| 进行研究 | 研究 |
| 进行分析 | 分析 |
| 开展探索 | 探索 |
| 实施推进 | 推进 |
| 加以运用 | 运用（视语境而定）|
| 作出贡献 | 贡献于…… |
| 取得成效 | 见效、收效 |
| 形成共识 | 共识形成 |
| 达成目标 | 实现目标 |

## 六、立场词

### 6.1 用

```
笔者认为 / 笔者以为 / 笔者主张
本研究主张
本研究观察到
本研究的判断是
```

### 6.2 不用

```
本文认为（书面化但仍 AI 感强）
研究表明（除非真的在引用他人研究）
我们认为（除多作者文章外，单作者论文中违和）
学界普遍认为（用了就要给出依据，否则属空洞修辞）
```

## 七、政治套话清单（学术语境内）

下列短语在学术论文中应避免，属于政策文件话语：

```
高质量发展
深化改革
完善体制机制
统筹推进
持续优化
进一步加强
不断完善
切实加强
扎实推进
有力推动
全面贯彻
深入贯彻
认真落实
着力解决
扎实开展
聚焦发力
精准发力
靶向发力
```

例外：在引用政策文本或讨论政策文件本身时可保留。但作者自己的论证语言中不应出现。

## 八、特殊词汇决议（本项目级）

如果用户的项目中曾就某词作过决议，记录在此供参照：

| 词 | 决议 | 原因 |
|---|---|---|
| 因果链 | 改为"分析链条" | 因果方向不严格单向，五案例中部分存在反向影响 |
| 国家能力 | 跨国比较中改为"政策工具组合与执行能力" | 涉台合规 |
| 国家 | 涉台语境删除 | 涉台合规 |
| 拔尖人才 | 改为"高层次工程科技人才" | 概念精度 |
| 超常布局 | 改为"战略性重构" | 学理化 |

新项目启动时建立项目级决议表，写入项目根目录的 `terminology.md`。

## 九、扫描脚本

```bash
#!/bin/bash
# 一键扫描全文禁用词
FILE=$1

echo "=== 硬禁用词 ==="
grep -nE '深度[^度学]|显著(?!性)|赋能|恰恰|重塑[^力]' "$FILE"

echo "=== 句首禁语 ==="
grep -nE '^[。\s]*(值得|不难|显而易见|综上|综合以上|总而言之|由此可见|不言而喻)' "$FILE"

echo "=== 套话 ==="
grep -nE '具有重要意义|新时代背景下|为.*提供.*参考|势在必行|迫在眉睫|前所未有|当务之急' "$FILE"

echo "=== 虚化动词 ==="
grep -nE '进行(研究|分析|探索|讨论)|开展(研究|探索|调研)|实施(推进|落实)' "$FILE"

echo "=== 立场词违例 ==="
grep -nE '研究表明|本文认为' "$FILE"

echo "=== 政治套话 ==="
grep -nE '高质量发展|深化改革|完善体制机制|统筹推进|持续优化|进一步加强|不断完善' "$FILE"

echo "=== 密度统计 ==="
echo "而非: $(grep -c '而非' "$FILE")  (上限 5)"
echo "不仅...更: $(grep -cE '不仅.{0,20}更' "$FILE")  (上限 2)"
echo "从.到: $(grep -cE '从.{1,10}到' "$FILE")  (上限 2)"
echo "本文/本节: $(grep -cE '本文|本节' "$FILE")  (全文 ≤ 5)"
```

保存为 `scan-banned.sh`，每次修改后运行一遍。
